# Ситуаційні картинки для карточок ПДР

> Карточки, де простий дорожній знак НЕ підходить — потрібна згенерована ілюстрація дорожньої ситуації.
> Для кожної ситуації є промпт для ChatGPT / DALL-E / Midjourney.
> Стиль: **вид зверху (top-down diagram), мінімалізм, чіткі стрілки напрямків, кольорові авто (червоний, синій, зелений)**

---

## 1. Перехрестя рівнозначних доріг (2-3 авто)

### 1.1. Два авто на рівнозначному — хто справа
**Карточки:** b14-equal-*, intersection-equal-right-*, b20-equal-road-*

```
Top-down diagram of an uncontrolled intersection of two equal roads.
Two cars: Red car approaching from the south going straight, Blue car approaching from the east going straight.
No traffic signs, no traffic lights. Clean white road markings.
Arrows showing direction of movement for each car.
Minimalist flat style, light gray road, white dashed center lines.
Label: "A" on red car, "B" on blue car.
```

### 1.2. Три авто на рівнозначному — порядок проїзду
**Карточки:** b14-equal-three-*, b20-3car-equal-*, b20-intersection-3car-*

```
Top-down diagram of an uncontrolled intersection with 4 roads.
Three cars: A (red) from north going straight, B (blue) from west going straight, C (green) from east turning left.
No traffic signs. Dashed center lines on all roads.
Arrows showing each car's direction. Letters A, B, C labeling each car.
Minimalist flat style, light gray asphalt.
```

### 1.3. Рівнозначне — один прямо, один ліворуч
**Карточки:** b20-equal-road-left-turn-*, b14-equal-left-right-*

```
Top-down intersection diagram. Two equal roads crossing.
Red car from south going straight. Blue car from west turning left (crossing red car's path).
No signs, no lights. Dashed lines. Curved arrow for left turn.
Minimalist flat style. Labels A and B.
```

## 2. Перехрестя нерівнозначних доріг (головна + другорядна)

### 2.1. Головна дорога прямо — другорядна справа
**Карточки:** b14-giveway-*, b14-main-*, intersection-main-vs-secondary-*

```
Top-down diagram of uncontrolled intersection. Main road goes west-east (horizontal, wider, bolder lines). Secondary road goes north-south.
Red car on main road going east (straight). Blue car on secondary road from south, has "Give Way" triangle sign (2.1).
Arrow: red straight, blue straight across.
Minimalist flat, main road visually thicker. Sign 2.1 icon near blue car.
```

### 2.2. Поворот ліворуч з головної — хто поступається
**Карточки:** b14-main-left-turn-*, b14-main-road-left-turn-*

```
Top-down intersection. Main road horizontal. Secondary road vertical.
Red car on main road turning LEFT from east. Blue car on secondary road going straight from south, has Give Way sign.
Curved left-turn arrow for red car. Straight arrow for blue.
Question: who yields? Labels A and B.
```

### 2.3. Т-подібне перехрестя — головна + другорядна
**Карточки:** b14-t-junction-*, b14-main-right-turn-secondary-*

```
Top-down T-junction. Main road horizontal. Dead-end road from bottom.
Red car from bottom going straight onto main road. Blue car on main road from left going right.
Give Way sign near red car. Arrows showing directions.
Minimalist, clear T-shape.
```

### 2.4. Головна дорога змінює напрямок
**Карточки:** b14-main-road-turns-right-*, b20-main-road-direction-change-*

```
Top-down intersection with sign "2.3 Main Road" showing main road turning right (diagonal arrow on sign).
Red car going straight (leaving main road). Blue car on the road where main continues (turning right).
Sign 2.3 with direction diagram shown. Arrows for each car.
Question: who has priority now?
```

```
Top-down intersection with traffic lights.
Traffic light showing: RED main circle + GREEN right arrow in additional section.
Red car wants to turn right on green arrow. Blue car from left has GREEN main signal going straight.
Arrows showing paths crossing. Traffic light icon clearly visible.
```

### 4.2. Поворот ліворуч на зелений — зустрічний прямо

```
Top-down intersection with traffic lights showing GREEN for both directions.
Red car turning LEFT. Blue car going STRAIGHT from opposite direction (oncoming).
Their paths cross. Both have green light.
Curved arrow for left turn, straight arrow for oncoming.
```

## 5. Обгін — ситуаційні схеми

### 5.1. Обгін — виїзд на зустрічну смугу
**Карточки:** b11-overtaking-oncoming-*, b11-overtaking-chain-*

```
Top-down view of two-lane road (one lane each direction). Dashed center line.
Red car (A) in right lane overtaking Blue truck (B) — red car partially in oncoming lane.
Green car (C) approaching from opposite direction in oncoming lane.
Arrows showing all three vehicles' directions. Danger zone highlighted.
```

### 5.2. Подвійний обгін заборонений
**Карточки:** b11-overtaking-double-*, b14-6-double-*

```
Three-lane road. Red car A overtaking Blue car B, and Green car C trying to overtake Red car A simultaneously.
All three in adjacent lanes. Arrows showing chain overtaking.
Red "X" or warning indicator showing this is prohibited.
```

### 5.3. Обгін біля пішохідного переходу
**Карточки:** b11-overtaking-pedestrian-crossing-*

```
Two-lane road with pedestrian crossing (zebra stripes).
Red car attempting to overtake Blue car at the crossing zone.
Pedestrian figure on the crossing. Red prohibition zone around crossing.
```

## Загальні інструкції для генерації

**Стиль для дорожніх ситуацій:**
- Вид зверху (top-down / bird's eye)
- Мінімалізм, плаский дизайн
- Дороги: світло-сірий асфальт, білі лінії розмітки
- Авто: кольорові прямокутники з буквами (A, B, C)
- Стрілки: чіткі, кольорові, показують напрямок руху
- Знаки: маленькі іконки біля відповідних авто
- Фон: білий або дуже світло-сірий
- Розмір: квадратний (1:1), 512x512 або 1024x1024

**Стиль для медичних ілюстрацій:**
- Простий медичний діаграмний стиль
- Чіткі лейбли та стрілки
- Не реалістичний — схематичний
- Кольори: синій/зелений для правильних дій, червоний для небезпеки

**Загальний промпт-префікс для всіх:**
```
Create a simple, clean, minimalist educational diagram for a Ukrainian driving test app.
Style: flat design, top-down view, light gray roads, colored cars (red=A, blue=B, green=C),
white road markings, directional arrows. No text except labels A, B, C.
Square format, white background. Professional driving school style.
```
